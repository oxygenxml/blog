Handling Translations

Radu Coravu
Read time: 1 minute(s)

Some useful advice for translating a DITA XML-based project can be found here: Translating your DITA Project.

The translated DITA content can also be stored on Git:
  • Keep the translation for each language in a separate folder in the same Git project. As a benefit, binary resources can be shared between projects.

  • Keep a separate parallel project with all the translations.
