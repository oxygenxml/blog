What is New in Oxygen AI Positron Assistant 5.0

Alin Belu
30 May 2025
Read time: 2 minute(s)

The new Oxygen AI Positron Assistant 5.0 brings meaningful improvements to help content creators use AI more effectively in structured authoring.

One of the most useful updates is the ability to generate or update files like XSLT, XSD, Schematron, JSON Schema, and DTD just by providing a brief description. You can then review changes using the built-in visual file comparison tool.

For DITA users, the assistant now supports advanced content transformation, making it easy to convert between document types or restructure topics using AI suggestions.

Other key updates include:

  • Eclipse plugin support

  • The option to exclude specific files from AI processing

  • Callback functions for validation or tool integration

  • Support for attaching Word, PowerPoint, and PDF files as input

  • Access to new reasoning models like o3, o1, o4-mini, and more

  • Integration with xAI Grok and other custom AI services through specialized connectors

Additional refinements improve comment resolution, validation fixes, and allow referencing external resources or prompts.
